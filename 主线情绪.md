## 亲密度收集器

### 数据结构

```lua
-- 属于玩家
IntimacyCollector = {
	LastCollectTime = 0
}
```

### 协议

```lua
-- 领取
handler = "IntimacyCollector" 
behavior = "Reward"

```

```lua
-- 刷新时间
handler = "IntimacyCollector" 
behavior = "Refresh"
参数：无
```
